## 기타 문법 설명
- try { ... } catch (Exception e) { ... } : 예외 처리 구문으로, 코드 실행 중에 발생할 수 있는 예외를 처리함. 예외가 발생하면 해당 예외의 메시지를 출력.
- JDBC(Java Database Connectivity): 자바에서 데이터베이스에 접속할 수 있도록 하는 자바 API
